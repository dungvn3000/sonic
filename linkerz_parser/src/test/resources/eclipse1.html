Eclipse Settings

Eclipse is really powerful and can save you making mistakes like missing @Override keyword on methods, manually removing unused imports, etc.

All these can be configured in eclipse per workspace basis.

Simply set the following settings and see the difference:-

In eclipse, Go To Windows → Preferences → Search for save and select Java Editor Save Actions
